===================
Install Anbox Cloud
===================

The guides in this section describe how to install Anbox Cloud.

.. note::
   There is a difference between the
   full Anbox Cloud installation and the Anbox Cloud Appliance (see
   :ref:`exp_anbox-cloud-variants`).
   This section focuses on **Anbox Cloud**. For instructions on how to
   install the **Anbox Cloud Appliance**, see :ref:`tut_installing-appliance`.

Make sure to check the
:ref:`requirements`
before you start your installation.
